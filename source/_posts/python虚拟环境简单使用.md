### 安装virtualenv
```
pip3 install virtualenv
```
### 查看版本
```
virtualenv --version
```
如果版本大于20，默认不复制系统库

### 进入项目文件夹，创建虚拟环境
```
virtualenv venv
```
### 进入环境
```
source venv/bin/activate
```
### 完成使用后退出虚拟环境
```
deactivate
```
