### 生成日期列表
```
# closed根据需要，可加可不加。
[_.strftime("%Y%m%d") for _ in pd.date_range(start='2020-09-02', end='2020-10-28', closed='left')]
```

### 获取今天日期
```
>>> from datetime import date
>>> date.today().strftime("%Y%m%d")
'20201028'

```
